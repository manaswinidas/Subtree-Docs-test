Introduction

Subtree-Docs-test

This is a test repository to check the syncing of GitHub wiki and GitBook content for documentation using git subtree.

Steps to reproduce

Add the subtree remote using:`git remote add -f kogito-wiki https://github.com/kiegroup/kogito-runtimes.wiki.git` Add the subtree using:`git subtree add --prefix=kogito-wiki https://github.com/kiegroup/kogito-runtimes.wiki.git master` Pull the latest updates from subtree:`git subtree pull-all`

Install gitbook dependencies

npm install

Build and run kogito-wiki gitbook in your local environment

Create the directories and files for a book from its SUMMARY.md file using:

gitbook init

You can serve the kogito-wiki repository as a gitbook using:

gitbook serve

Or simply build the static website using:

gitbook build
